SIMULADORES


Os simuladores recomendados para as práticas são:

QUCS - Quite Universal Circuit Simulator

Versão 0.0.19 // 22 de janeiro de 2017
O QUCS é um simulador de circuitos que é um FOSS (Free Open Source Software) quer dizer, gratuíto e seu código fonte é aberto. O aplicativo está todo traduzido para o idioma português.

Como abrir o programa: Para rodar o programa no Windows, faça o download do arquivo chamado qucs-0.0.19-win32-mingw482-asco-freehdl-adms.zip  que contém todas os componentes que dependem para seu funcionamento, assim apenas precisa descompactar a pasta e clicar no arquivo qucs para executar (programa do tipo portable que não precisa ser instalado) . Nas várias distribuições Linux, é compatível com o Wine (camada de compatibilidade com Windows), basta ir até pasta no terminal e digitar "wine qucs" para executar. Para usuários do sistema MacOSX 10.17, um  arquivo do tipo pkg está disponível para download. Esta versão nova é bem atualizada porque o programa passou a usar um algoritmo de simulação chamado ASCO e uma implementação de linguagem de hardware chamada freeHDL.

Versão 0.0.15 //25 de abril de 2009
Esta versão é mais antiga, muito leve e estável para um bom desempenho em computadores de dez anos atrás. Download 5,6MB para Windows 32x (compatível com Wine) .

PROTEUS Design Suite

Versão 8.7 // 28 de fevereiro de 2017
Este programa é pago pois atende ao mercado de profissionais da área de eletrônica. Mas se quiser gratuitamente usar a versão Trial para estudantes, há algumas limitações e  não dá pra salvar projetos. Mas é boa para experimentar para o aprendizado seus recursos avançados como simulações animadas.

Versão 7.9 // 5 Mar 2008
Legado dessa versão é da época 7x do Proteus, além de ser mais leve, sua compatibilidade com Wine funciona muito bem. O Proteus consiste de dois módulos chamados ISIS e ARES, que são as diferentes partes do simulador.

Simuladores Online (Software na "Nuvem")

Podemos recomendar dois: EasyEDA e CircuitLab que rodam diretamente no seu browser.
